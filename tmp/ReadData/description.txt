Synopsis: This will read an incoming data string, search for a matching expression and return the expression.

Input param:  
data = Incoming data from a wfv to search for
expression = An expression to match on and return.

Output param:  status
Output values:  Either an empty string or the return value specified in the parentheses

Sample Params Config (fill in over on the left): 
{
"data":"message",
"expression":"(High CPU)"
}
